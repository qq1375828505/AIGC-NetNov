# 屏幕注册

> **状态：✅ 方案已确认** | 统一决策见 `Operit网文写作改造计划.md` 5.3 节
> - 网文写作模块使用 **ToolPkg 插件层**注册（不修改原生层）。
> - 原生层方案保留为备选，用于需要自定义侧边栏分组标题"网文写作"的场景。
> - 7 个屏幕统一通过 ToolPkg API 注册到 `main_sidebar_plugins` 分组。

---

## Operit 注册机制概述

Operit 有两套并行的注册机制：

### 机制 A：原生层（Kotlin）

| 文件 | 作用 |
|------|------|
| `NavItem.kt` | sealed class，定义导航项 |
| `OperitScreens.kt` | `sealed class Screen(navItem, titleRes, ...)`，定义屏幕 |
| `ScreenRouteRegistry.kt` | 通过**反射自动发现** Screen 子类（`Screen::class.java.declaredClasses`），无需手动注册 Screen 本身 |
| `ScreenRouteRegistry` 的 `hostEntryDefinitions` | 手动指定每个 Screen 的 `entryId`、`surface`、`launchNavItem`、`icon`、`order` |
| `NavigationSurface` 枚举 | `MAIN_SIDEBAR_AI`, `MAIN_SIDEBAR_TOOLS`, `MAIN_SIDEBAR_PLUGINS`, `MAIN_SIDEBAR_SYSTEM`, `TOOLBOX` |

### 机制 B：ToolPkg 插件层（JS/TS）

| API | 作用 |
|-----|------|
| `ToolPkg.registerNavigationEntry({id, route, surface, title, icon, order})` | 注册侧边栏入口 |
| `ToolPkg.registerUiRoute({id, route, runtime, screen, params, title})` | 注册 UI 路由 |

- surface 为字符串，如 `"main_sidebar_plugins"`
- 参考示例：`examples/sidebar_account_book/src/main.ts`

---

## 推荐方案：ToolPkg（主方案）

网文写作模块推荐使用 ToolPkg 方案，**不修改原生层代码**。所有注册在 ToolPkg 的 `main.ts` 中完成。

### main.ts 示例

```typescript
// packages/novelide/src/main.ts

import { Icons } from "@operit/toolpkg";

export function registerToolPkg(): boolean {

  // ========================================
  // 注册 7 个侧边栏入口
  // ========================================

  ToolPkg.registerNavigationEntry({
    id: "novel_works_sidebar",
    route: "toolpkg:com.operit.novelide:ui:novel_works",
    surface: "main_sidebar_plugins",
    title: { zh: "我的作品", en: "My Works" },
    icon: Icons.Book,
    order: 120,
  });

  ToolPkg.registerNavigationEntry({
    id: "novel_editor_sidebar",
    route: "toolpkg:com.operit.novelide:ui:novel_editor",
    surface: "main_sidebar_plugins",
    title: { zh: "写作编辑器", en: "Novel Editor" },
    icon: Icons.Edit,
    order: 130,
  });

  ToolPkg.registerNavigationEntry({
    id: "novel_materials_sidebar",
    route: "toolpkg:com.operit.novelide:ui:novel_materials",
    surface: "main_sidebar_plugins",
    title: { zh: "资料管理", en: "Materials" },
    icon: Icons.FolderSpecial,
    order: 140,
  });

  ToolPkg.registerNavigationEntry({
    id: "novel_outline_sidebar",
    route: "toolpkg:com.operit.novelide:ui:novel_outline",
    surface: "main_sidebar_plugins",
    title: { zh: "大纲管理", en: "Outline" },
    icon: Icons.List,
    order: 150,
  });

  ToolPkg.registerNavigationEntry({
    id: "novel_stats_sidebar",
    route: "toolpkg:com.operit.novelide:ui:novel_stats",
    surface: "main_sidebar_plugins",
    title: { zh: "写作统计", en: "Statistics" },
    icon: Icons.BarChart,
    order: 160,
  });

  ToolPkg.registerNavigationEntry({
    id: "novel_tools_sidebar",
    route: "toolpkg:com.operit.novelide:ui:novel_tools",
    surface: "main_sidebar_plugins",
    title: { zh: "写作工具", en: "Tools" },
    icon: Icons.AutoFixHigh,
    order: 170,
  });

  ToolPkg.registerNavigationEntry({
    id: "novel_workspace_sidebar",
    route: "toolpkg:com.operit.novelide:ui:novel_workspace",
    surface: "main_sidebar_plugins",
    title: { zh: "工作区", en: "Workspace" },
    icon: Icons.Workspaces,
    order: 180,
  });

  // ========================================
  // 注册 7 个 UI 路由
  // ========================================

  ToolPkg.registerUiRoute({
    id: "novel_works",
    route: "toolpkg:com.operit.novelide:ui:novel_works",
    runtime: "compose_dsl",
    screen: novelWorksScreen,
    params: {},
    title: { zh: "我的作品", en: "My Works" },
  });

  ToolPkg.registerUiRoute({
    id: "novel_editor",
    route: "toolpkg:com.operit.novelide:ui:novel_editor",
    runtime: "compose_dsl",
    screen: novelEditorScreen,
    params: { workId: "string" },
    title: { zh: "写作编辑器", en: "Novel Editor" },
  });

  ToolPkg.registerUiRoute({
    id: "novel_materials",
    route: "toolpkg:com.operit.novelide:ui:novel_materials",
    runtime: "compose_dsl",
    screen: novelMaterialsScreen,
    params: { workId: "string" },
    title: { zh: "资料管理", en: "Materials" },
  });

  ToolPkg.registerUiRoute({
    id: "novel_outline",
    route: "toolpkg:com.operit.novelide:ui:novel_outline",
    runtime: "compose_dsl",
    screen: novelOutlineScreen,
    params: { workId: "string" },
    title: { zh: "大纲管理", en: "Outline" },
  });

  ToolPkg.registerUiRoute({
    id: "novel_stats",
    route: "toolpkg:com.operit.novelide:ui:novel_stats",
    runtime: "compose_dsl",
    screen: novelStatsScreen,
    params: {},
    title: { zh: "写作统计", en: "Statistics" },
  });

  ToolPkg.registerUiRoute({
    id: "novel_tools",
    route: "toolpkg:com.operit.novelide:ui:novel_tools",
    runtime: "compose_dsl",
    screen: novelToolsScreen,
    params: {},
    title: { zh: "写作工具", en: "Tools" },
  });

  ToolPkg.registerUiRoute({
    id: "novel_workspace",
    route: "toolpkg:com.operit.novelide:ui:novel_workspace",
    runtime: "compose_dsl",
    screen: novelWorkspaceScreen,
    params: {},
    title: { zh: "工作区", en: "Workspace" },
  });

  return true;
}
```

## 备选方案：原生层（Kotlin）

如果需要自定义侧边栏分组标题为"网文写作"（而非默认的"插件"分组），则需要修改原生层。

### 步骤 1：NavItem.kt 追加

文件路径：`app/src/main/java/com/ai/assistance/operit/ui/common/NavItem.kt`


### 步骤 2：OperitScreens.kt 追加

文件路径：`app/src/main/java/com/ai/assistance/operit/ui/main/screens/OperitScreens.kt`

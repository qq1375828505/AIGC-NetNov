## 核心数据流

```
用户输入（WebView HTML 前端）
        │
        ▼
┌─────────────────────────────┐
│  bridgeScript 拦截/覆盖      │  例如：把原 127.0.0.1:4590 调用路由到 NativeBridge
└─────────────────────────────┘
        │
        ▼
┌─────────────────────────────┐
│     WebUIBridge（Kotlin）    │
└─────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│  分支 1：普通聊天消息                                          │
│  sendChatMessage(text, sessionId)                             │
│       │                                                       │
│       ▼                                                       │
│  ChatViewModel.sendTextMessage() ──► 主 AI 生成回复           │
│       │                                                       │
│       ▼                                                       │
│  pushAIResponse / pushAIStreamChunk 推送回前端显示            │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│  分支 2：工具调用                                              │
│  AI 判断需要工具 ──► ToolExecutor 执行                        │
│       │                                                       │
│       ├── 读文件 / 读数据库 ──► 返回结果给 AI                 │
│       ├── 写文件 / 写数据库 ──► 持久化后返回                  │
│       ├── 联网搜索 ──► DuckDuckGo API                         │
│       ├── delegate_to_sub_agent ──► 子 Agent 多轮迭代         │
│       └── run_workflow ──► 按步骤调用工具                     │
│       │                                                       │
│       ▼                                                       │
│  最多 5 轮工具迭代后返回最终回复                               │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│  分支 3：语音通话                                              │
│  openVoiceCall() ──► VoiceCallManager.startCall()             │
│       │                                                       │
│       ├── SpeechService 持续识别用户语音                      │
│       ├── 识别到完整语句 ──► ChatViewModel.sendTextMessage()  │
│       ├── AI 回复文本 ──► VoiceService.speak() 播放           │
│       └── 结束通话 ──► 完整转写生成总结并注入聊天             │
└─────────────────────────────────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────────────────────────────────┐
│  分支 4：文件/设置操作                                         │
│  readFile / writeFile / listFiles ──► SAF + 本地文件系统     │
│  getSetting / setSetting ──► SharedPreferences                │
└─────────────────────────────────────────────────────────────┘
```

## 关键文件清单

```
app/src/main/java/com/aificide/app/ui/main/OperitAppV2.kt
  └── 主入口，初始化 WebUI 或原生 Compose 界面

app/src/main/java/com/aificide/app/ui/webui/WebUIContainer.kt
  └── WebView 容器，注入 bridgeScript，管理 ActivityLauncher/权限

app/src/main/java/com/aificide/app/ui/webui/WebUIBridge.kt
  └── JS ↔ Kotlin 桥接，暴露 Native 方法给前端

app/src/main/java/com/aificide/app/ui/webui/voice/VoiceCallManager.kt
  └── 语音通话管理器：ASR + TTS + 主 AI + 自动总结

app/src/main/java/com/aificide/app/ui/features/chat/viewmodel/ChatViewModel.kt
  └── 主聊天 ViewModel，协调发送、历史、状态

app/src/main/java/com/aificide/app/services/core/ChatHistoryDelegate.kt
  └── 会话历史管理：创建/切换/删除/置顶

app/src/main/java/com/aificide/app/api/speech/SpeechService.kt
app/src/main/java/com/aificide/app/api/speech/SpeechServiceFactory.kt
  └── 语音识别接口与工厂

app/src/main/java/com/aificide/app/api/voice/VoiceService.kt
app/src/main/java/com/aificide/app/api/voice/VoiceServiceFactory.kt
  └── 语音合成接口与工厂

app/src/main/assets/web/ui/完整版.html
  └── WebView 运行时的前端单页应用

app/src/main/java/com/aificide/app/data/repository/ChatHistoryManager.kt
app/src/main/java/com/aificide/app/data/db/ChatDao.kt
  └── SQLite 数据访问与会话管理

app/src/main/java/com/aificide/app/data/services/AiService.kt
  └── 统一 AI API 调用，多厂商适配

app/src/main/java/com/aificide/app/data/services/agent/ToolExecutor.kt
  └── 内置工具执行器

app/src/main/AndroidManifest.xml
  └── RECORD_AUDIO 等权限声明
```

## DrawerContent 修改

在 `DrawerContent.kt` 中添加分组标题：

```kotlin
// 在 DrawerContent.kt 的 LazyColumn 中添加

// 原有 AI 助手分组
item { SectionHeader(stringResource(R.string.section_ai_assistant)) }
items(navItems.filter { it in aiAssistantItems }) { item ->
    NavItemRow(item = item, ...)
}

// 原有工具分组
item { SectionHeader(stringResource(R.string.section_tools)) }
items(navItems.filter { it in toolItems }) { item ->
    NavItemRow(item = item, ...)
}

// ====== 新增网文写作分组 ======
item { SectionHeader(stringResource(R.string.section_novel_writing)) }
items(navItems.filter { it in novelWritingItems }) { item ->
    NavItemRow(item = item, ...)
}

// 原有系统分组
item { SectionHeader(stringResource(R.string.section_system)) }
items(navItems.filter { it in systemItems }) { item ->
    NavItemRow(item = item, ...)
}
```

📜 用户协议
用户协议 & 隐私政策

1. 服务条款

欢迎使用 AI-Fic-IDE（以下简称"本软件"）。本软件基于 GNU LGPLv3 协议开源，您可以自由使用、修改和分发。

2. 数据隐私

本软件完全本地运行，您的写作数据、AI 对话记录均存储在本地设备。我们不会收集任何个人数据。

本地模型推理完全离线进行。如使用 API 模式，数据将通过您配置的 API 端点传输。
